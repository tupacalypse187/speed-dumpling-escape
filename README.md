## 🎮 Features

### ⚡ Core Loop
- **Treadmill training** — run on hub treadmills to gain speed every 0.4s, with combo-pop "+N" numbers, steam puffs, and satisfying ticks.
- **Infinite levels** — thresholds at 100 / 300 / 750 / 1.5K, then ×2.2 forever (rounded to nice numbers). Your dumpling changes color and earns procedural names past Gold.
- **Milestone skins** — ✨ Sparkle Aura (Lv 5), 🔥 Flame Trail (Lv 10), 🌈 Rainbow Cycling (Lv 15), 🌌 Galaxy Starfield (Lv 20).
- **Speed = power** — higher speed means faster running and longer jumps. That's why you train.

### 🚪 Obbies
- **Door-per-level arcade** — every level you unlock opens a matching color-themed door in the hub (two-row arcade with an elevated deck as it grows).
- **Endless procedural obbies** — Obby 4+ are generated by a seeded, deterministic algorithm: same obby number = same course forever. Win rewards double (25, 50, 100…).
- **Fair-by-construction difficulty** — gaps use 55–70% of your jump capability at the required level, machine-verified by `verifyObby()`. Challenge comes from spikes, movers and timing, never barely-makeable jumps.
- **Breathing landing pads** — teal pads that grow and shrink (60%–130%) on gentle sine cycles with rippling glow. Collision tracks the animation every frame.
- **Moving platforms, spikes, lava** — ride sine-wave movers, weave spike beds, and don't touch the bubbling lava.
- **🎯 Practice mode** — a no-pressure training course with checkpoint flags and labeled drill sections. *(see roadmap)*

### ⭐ Weekly Challenge
- A golden door with a spinning star and a live countdown.
- ISO-week-seeded course that rotates every Monday — same week, same course.
- Big first-clear coin payout, small repeat payout, per-week best-time history in the Trophy Room.

### 🌀 Rebirths
- Unlock at Level 5. Reset your speed for a **permanent ×2 speed-gain multiplier** per rebirth (stacks with upgrades).
- Full ceremony: slow-mo, white flash, confetti storm, gong + choir.
- A permanent colored halo ring orbits your dumpling, hue-shifting with each rebirth.

### 🐾 Pets
Five unlockable companions with springy follow physics and passive bonuses:
| Pet | Bonus | Unlock |
| --- | --- | --- |
| 🐶 Dumpling Pup | +10% speed gain | 10 wins |
| 💨 Steam Sprite | auto-collects nearby treadmill ticks | reach Lv 3 |
| 🥟 Mini Wonton | +15% jump power | beat Obby 2 |
| 🌶️ Chili Imp | +20% coins | 5 achievements |
| 🐉 Soup Dragon | +50% speed gain (segmented body!) | 1 rebirth |

### 🪙 Coins & Shop
- Coin pickups scattered (and hidden in risky spots) through every obby + completion bonuses.
- Spend at the hub shop: **hats** (steamer lid, chef, party, crown), **trail color packs**, and stackable **+5% speed charms**.

### 🏆 Meta
- **12 achievements** with toast popups (Speed Demon, Obby Master, Marathon…).
- **Trophy Room** — lifetime stats, per-obby best times, weekly history, achievement gallery.
- **NEW RECORD** celebrations with sound + toast when you beat a best time.
- **Ghost replays** — race a translucent replay of your best run. *(see roadmap)*

### 🎨 Presentation
- 100% canvas-rendered dumpling: squash-and-stretch, blinking, scurrying feet, animated pleats, hats, milestone skins, rebirth halo.
- Layered parallax world: mountains → steamer baskets → clouds → dumpling hills → swaying grass, plus god rays, dust motes and a warm day-cycle tint.
- **100% synthesized audio** (Web Audio API): all SFX + a chiptune BGM loop. No audio files.
- Screen shake, speed lines, landing rings, camera lookahead + speed zoom-out — juice everywhere.

## 🕹️ Controls

| Key | Action |
| --- | --- |
| `A` / `D` or `←` / `→` | Move |
| `Space` / `W` / `↑` | Jump (coyote time + jump buffer included) |
| `E` | Interact — doors, shop, shrine, button |
| `Tab` / `P` | Pets panel |
| `R` | Exit current obby |
| `Esc` | Pause menu (volume, shake, particles, ghost toggle) |
| `M` | Mute |

## 🚀 Getting Started

```bash
npm install       # install dependencies
npm run dev       # start dev server (http://localhost:3000)
npm run build     # type-check + production build
npm run preview   # preview the production build
```

Requires Node 20+.

## 💾 Save Data

Progress lives in your browser's `localStorage` (speed, wins, coins, pets, rebirths,
records, ghosts, settings). The schema is versioned and migrates gracefully — old saves
keep working across updates. Reset anytime from the title screen or pause menu.
